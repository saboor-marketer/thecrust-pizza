# The Crust Pizza Website

A modern, responsive, and professional website for The Crust Pizza restaurant, built with HTML5, CSS3, Vanilla JavaScript, and Bootstrap 5.

## 🎨 Design Features

- **Color Palette**: Red and orange following the 60-30-10 rule
  - 60% White/Light Gray (Backgrounds)
  - 30% Red (#E63946) (Primary Color)
  - 10% Orange (#F4A261) (Accent Color)
- **Modern UI/UX**: Clean, minimalistic, and professional design
- **Responsive**: Fully responsive design that works on all devices
- **Animations**: Smooth hover effects and transitions throughout

## 🚀 How to Use

1. **Open the Website**
   - Simply double-click `index.html` to open it in your web browser
   - No server or installation required

2. **Navigate the Site**
   - Use the navigation bar to move between sections
   - Click on menu items to scroll smoothly to sections

3. **Browse the Menu**
   - Use category tabs to filter menu items
   - Hover over items to see effects
   - Click "Order Now" for direct WhatsApp ordering
   - Click "Add to Cart" to add items to your cart

4. **Manage Your Cart**
   - Click the cart icon in the navigation bar
   - Review selected items
   - Remove items if needed
   - Click "Checkout via WhatsApp" to place your order

5. **Contact Information**
   - Visit the Contact section for location details
   - Use the floating WhatsApp button for quick contact
   - Follow social media links for updates

## 🔧 Customization

To customize the website:

1. **Colors**: Modify CSS variables in `styles.css` under `:root`
2. **Menu Items**: Edit the `menuItems` array in `script.js`
3. **Reviews**: Update the `reviews` array in `script.js`
4. **News**: Modify the `news` array in `script.js`
5. **Contact Info**: Update the contact section in `index.html`
